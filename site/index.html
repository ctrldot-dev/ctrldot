Ctrl Dot

Runtime execution boundary for AI agents.

A local daemon that sits between agents and their actions. Budget enforcement, loop detection, resolution gating, deterministic STOP, and signed run artefacts.

  • Budget enforcement
  • Loop detection
  • Resolution gating
  • Deterministic STOP
  • Signed run artefacts

Quickstart

git clone https://github.com/ctrldot-dev/ctrldot.git
cd ctrldot
make build-ctrldot
./bin/ctrldot daemon start

See the setup guide for full install, adapters (OpenClaw, CrewAI), and smoke tests.
