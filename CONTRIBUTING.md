# Contributing

- Run `go test ./...` before opening a PR.
- Use `gofmt` on Go code (CI checks `go fmt ./...`).
- Keep API changes backward compatible where possible.
- Include stable reason codes for new DENY/STOP paths.

See [README.md](README.md) for build and run instructions, and [docs/SETUP_GUIDE.md](docs/SETUP_GUIDE.md) for full setup.
