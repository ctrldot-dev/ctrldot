# CrewAI + Ctrl Dot

CrewAI integration with Ctrl Dot is provided by the **adapter** in this repo.

→ **[adapters/crewai/README.md](../../adapters/crewai/README.md)** — install, `CtrlDotClient`, `CtrlDotLLM`, `CtrlDotToolWrapper`, and example.

Quick links:

- Adapter: `adapters/crewai/` (Python package)
- Example: `adapters/crewai/examples/crew_minimal.py`
- Setup: `make setup-crewai` then `source .venv/bin/activate` and run the example
